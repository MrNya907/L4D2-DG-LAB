# L4D2-DG-LAB
<p>求生之路2与郊狼联动插件</p>
<p>通过与<a href="https://github.com/hyperzlib/DG-Lab-Coyote-Game-Hub">郊狼游戏控制器——战败惩罚</a>项目的联动执行相应的处罚（感谢此项目的开发方便后期接入使用）</p>
<p>需要自行安装sourcemod插件（https://www.sourcemod.net/downloads.php?branch=stable）且在属性里添加-insecure</p>
<p>相应文件丢入相应位置，先运行上面的项目连接好郊狼之后进游戏，首次进游戏会生成配置文件在游戏根目录left4dead2\cfg\sourcemod\l4d2_dglab_v2.cfg里面可以修改强度等</p>
<p>暂不支持云服务器（会获取不到强度，后期可能会修）</p>
